# Terminal Card

## Install 
```bash
# install dependencies
npm install 

# Install globally 
npm install -g .
```

## Usage
Then you can use the terminal-card anywhere, such as:
```bash
# add a new term
terminal-card --at [term] --ad [definition]

# play games
terminal-card --game
```

## Uninstall
```bash
npm uninstall -g terminal-card
```
